Eyes Exercise

Description

This is another educational project for further mtml/css/js improvements. 2 eyes will follow any mouse position on the screen and they will not get tired at all.


Installation

To run this just download whole folder eyesExample and open the index.html file.


Usage

This is educational project for testing mtml/css/js manipulation.


Support

No support is provided with this application.


Roadmap

This fun app allows for many extra add-ons:

  • We can add other pairs of eyes with different size, shape, and even Cyclopes.
  • We can make part of them follow mouse, other part follow opposite direction and other parts just move independently.
  • We can make them change color or size based on certain conditions: amount of rotations, time since spawn etc.
