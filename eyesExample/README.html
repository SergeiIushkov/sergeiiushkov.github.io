Eyes Exercise

Description

This is another educational project for further mtml/css/js improvements. 2 yese will follow any mouse position and they will not get tired at all.


Installation

To run this just download whole folder eyesExample and open the index.html file.


Usage

This is educational project for testing mtml/css/js manipulation.


Support

No support is provided with this application.
